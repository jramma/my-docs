# Docs with Starlight

Documentation site built with [Astro Starlight](https://starlight.astro.build/).

## Quick start

```bash
pnpm i
pnpm dev
```

## How it works

- Pages live in `/content/docs`
- Use `.md` for content, `.mdx` for content with components
- Builds to static HTML

## Learn more

- [Starlight docs](https://starlight.astro.build/)
- Check `astro.config.mjs` and `content.config.ts` to understand the setup
